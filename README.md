# Peace Blur Detector

## Installation

Clone this repository:

```bash
git clone https://github.com/Fatz-Dev/pyhton-trend-blur.git
cd peace-blur-detector
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## Run

```bash
python blur.py
```

Press `ESC` to close the application.

## Tech Stack

* Python
* OpenCV
* MediaPipe
